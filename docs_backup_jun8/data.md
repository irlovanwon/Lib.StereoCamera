# Data for Communication

| Data Type | Description |
|-----------|-------------|
| Stereo Images | One or multiple image pairs captured from left and right cameras |
| Depth Maps | Per-pixel depth information derived from stereo images |
| Point Cloud | 3D spatial data (X, Y, Z coordinates) |
| IMU Data | Inertial measurement unit data (acceleration, angular velocity) |
| Disparity Map | Pixel-level displacement between left and right images |
| Confidence / Error Maps | Per-pixel confidence level for depth values, allowing filtering of unreliable data |
| Temperature | Temperature readings from onboard sensors (IMU, barometer, left/right image sensors) |
